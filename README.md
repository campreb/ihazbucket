# I Haz Bucket
A simple S3 management tool

---

## Development Setup

Install Ruby Gems
```
bundle install
```

Copy .env file and configure with AWS credentials, region and bucket.
```
cp .env.sample .env
```

Start the server with rackup
```
rackup -p 4567
```
